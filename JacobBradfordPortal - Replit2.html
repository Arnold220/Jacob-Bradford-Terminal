    📋 Krok po kroku - GitHub:
    Krok 1: Utwórz konto GitHub
    1. Idź na github.com
    2. Kliknij "Sign up" (zielony przycisk w prawym górnym rogu)
    3. Wpisz email, hasło, username
    4. Potwierdź konto przez email
    Krok 2: Utwórz nowe repository
    1. Po zalogowaniu kliknij zielony przycisk "New" (po lewej stronie)
    2. W polu "Repository name" wpisz: jacob-bradford-terminal
    3. Zostaw zaznaczone "Public" (musi być public dla darmowego hostingu)
    4. Zostaw inne opcje domyślne
    5. Kliknij zielony przycisk "Create repository"
    Krok 3: Dodaj plik
    1. Na nowej stronie repository kliknij "uploading an existing file" (link na środku strony)
    2. Przeciągnij plik index.html z folderu static-export na stronę
      • LUB kliknij "choose your files" i wybierz plik
    3. W polu "Commit changes" na dole wpisz: Add terminal website
    4. Kliknij zielony przycisk "Commit changes"
    📋 Krok po kroku - Netlify:
    Krok 4: Deploy na Netlify
    1. Idź na netlify.com
    2. Kliknij "Sign up" i wybierz "Sign up with GitHub"
    3. Zaloguj się przez GitHub (pozwól na dostęp)
    4. Kliknij "New site from Git"
    5. Kliknij "GitHub"
    6. Znajdź i kliknij repository "jacob-bradford-terminal"
    7. Zostaw wszystko domyślnie i kliknij "Deploy site"
    Krok 5: Gotowe!
    • Netlify da Ci link typu: https://amazing-name-123456.netlify.app
    • Strona będzie dostępna za 1-2 minuty
    🔄 Jeśli GitHub się buguje:

    Alternatywna metoda - Upload bezpośrednio:

    1. Na stronie głównej repository
    2. Kliknij "Add file" → "Upload files"
    3. Przeciągnij index.html
    4. Commit changes
